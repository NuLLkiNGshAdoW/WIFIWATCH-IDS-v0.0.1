WIFIWATCH IDS
=============

Кішігірім желілік мониторинг қолданбасы (Windows Forms) — ARP/ICMP/порт скан, mDNS және опционалды трафик салу (sniffer).

Талаптар
--------
- .NET 8 (net8.0-windows)
- Visual Studio 2022/2026 или `dotnet` SDK
- (Опционально) Npcap — для сниффера

Зависимости (NuGet)
--------------------
- Newtonsoft.Json (13.0.4)
- System.Text.Encoding.CodePages (10.0.5)
- SharpPcap (5.1.0)
- PacketDotNet (1.0.3)

Сборка и запуск
---------------
1. Восстановите пакеты: `dotnet restore`
2. Соберите: `dotnet build`
3. Запустите из Visual Studio или `dotnet run`.

Важно: сниффер требует прав администратора и установленного Npcap. По умолчанию проект компилируется с символом `NOSNIFF`, который отключает код сниффера для безопасной сборки без Npcap.

API ключи и секреты
-------------------
- В `MainForm.cs` использован placeholder ключ для Gemini API. **Не** коммитьте реальные ключи в репозиторий.
- Рекомендуется использовать переменные окружения или `dotnet user-secrets`.

Рекомендации перед пушем
------------------------
1. Удалите строку, игнорирующую `MainForm.cs` из `.gitignore`, если вы переместили ключи в переменные окружения.
2. Убедитесь, что `NOSNIFF` не определён в `diplom2.csproj`, если вы хотите включить сниффер.
3. Добавьте `LICENSE` (например, MIT) если публикуете публично.

Примеры
-------
- Чтобы включить сниффер: откройте `diplom2.csproj`, удалите/измените `DefineConstants` содержащий `NOSNIFF`, убедитесь что Npcap установлен и запустите от администратора.
